# Rocket Boilerplate 
:hammer_and_pick: 

This boilerplate used [Rocket](https://rocket.rs/) + [Diesel](https://diesel.rs/) + [PostgreSQL](https://www.postgresql.org).

## How to run

```shell
git clone https://github.com/NikoManiac/rust-rocket-boilerplate.git
cd rust-rocket-boilerplate
cargo run
```
